Change Log

All notable changes to this project will be documented in this file. See standard-version for commit guidelines.

1.0.0 (2018-08-16)

Features

  • add rule specific directives for convenience and better semantic (f24763b)
  • deprecate forRoot for better convenient usage (c95dbcf)
  • support anyOf rule for full mapping with set operations (9e61bd9)

BREAKING CHANGES

  • The signature of ValidateResult is changed to be an interface

0.9.0 (2018-06-01)

Features

  • upgrade to Ng6 and Rxjs 6 and the project template (603c1a5)

0.8.2 (2017-12-26)

0.8.1 (2017-11-23)

0.8.0 (2017-11-18)

Features

  • deps: upgrade deps to support Angular 5 (d593fa8)

0.7.1 (2017-10-26)

Bug Fixes

  • rxjs: remove the ignored throw operator (96de94f)

0.7.0 (2017-10-26)

Features

  • rxjs: upgrade rxjs and use its new lettable feature (99440e4)

BREAKING CHANGES

  • rxjs: previously we added the operatators to Observable's prototype, now we don't. This could raise some missing references error at the user end.

Bug Fixes
